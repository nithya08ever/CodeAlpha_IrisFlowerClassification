# Iris Flower Classification

## Project Description

This project classifies Iris flowers into three species:

- Setosa
- Versicolor
- Virginica

using Machine Learning with the K-Nearest Neighbors (KNN) algorithm.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Dataset

Iris Dataset

## How to Run

Install dependencies

pip install -r requirements.txt

Run

python main.py

## Output

- Pair Plot
- Confusion Matrix
- Classification Report
- Accuracy Score
